# semantic-release

[![Build](https://github.com/gsmlg-ci/semantic-release/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/semantic-release/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/semantic-release`
- `ghcr.io/gsmlg-dev/semantic-release`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/semantic-release:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/semantic-release:latest
```

## Build

```bash
docker build -t gsmlg/semantic-release .
```
